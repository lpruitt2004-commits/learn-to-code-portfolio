# Glossary: Key Terms

- **Algorithm:** A step-by-step procedure for solving a problem.
- **Array:** An ordered list of values.
- **Boolean:** A data type with two values: true or false.
- **Commit:** A saved change in version control (Git).
- **Function:** A reusable block of code that performs a task.
- **Object:** A collection of key-value pairs in JavaScript.
- **Repository:** A storage space for code, often managed with Git.
- **Variable:** A named value that can change during program execution.
- **Branch:** A parallel version of a repository in Git.
- **Merge:** Combining changes from different branches in Git.
