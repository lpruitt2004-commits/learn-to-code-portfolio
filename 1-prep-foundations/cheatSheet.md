# Cheat Sheet: Quick Reference

## JavaScript
- Declare variable: `let x = 5;`
- Function: `function add(a, b) { return a + b; }`
- Array: `let arr = [1,2,3];`
- Object: `let obj = { key: 'value' };`

## Python
- Variable: `x = 5`
- Function: `def add(a, b): return a + b`
- List: `arr = [1,2,3]`
- Dict: `obj = { 'key': 'value' }`

## Git
- Init repo: `git init`
- Add: `git add .`
- Commit: `git commit -m "msg"`
- Push: `git push`
- Status: `git status`

## CLI
- List files: `ls`
- Change dir: `cd folder`
- Make dir: `mkdir folder`
- Create file: `touch file.txt`
