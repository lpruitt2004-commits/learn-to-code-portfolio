# Fundamental Python

Python is a versatile language used for web development, data science, automation, and more.

## Syntax and Variables
```python
name = "Alex"  # string
age = 25        # integer
is_student = True  # boolean
```

## Functions
```python
def greet(name):
    return f"Hello, {name}!"
print(greet("World"))
```

## Data Structures
```python
fruits = ["apple", "banana", "cherry"]
person = {"name": "Sam", "age": 30}
```

## Scripting Basics
```python
for i in range(3):
    print(i)
```
