# Assignments: Prep & Foundations

Below are assignments for each topic. Submit your solutions in the corresponding files. CLxL2 can review and provide feedback!

---

## 1. Intro to Software Engineering
- Write a short essay (100-200 words): "What does a software engineer do?"
- List 3 habits of effective engineers.

## 2. Web Dev Prep Course
- Draw (or describe) the path a browser takes to load a web page.
- List 3 browser dev tools and their uses.

## 3. Fundamental JavaScript
- Write a function that returns the sum of two numbers.
- Create an array of your 3 favorite websites and print each one.

## 4. Fundamental Python
- Write a Python script that prints "Hello, world!"
- Create a list of 5 numbers and print their average.

## 5. Command Line Basics
- List 5 common bash commands and what they do.
- Write a command to create a new directory and a file inside it.

## 6. Version Control Basics
- Describe the difference between git add, git commit, and git push.
- Initialize a new git repo and make your first commit (show the commands).

## 7. Workflow and Mindset
- Write 3 tips for staying productive while learning to code.
- Describe a time you solved a problem by asking for help.

> Place your answers in topic-specific files (e.g., fundamentalJavaScript.md, fundamentalPython.md, etc.) or create new files as needed. CLxL2 can grade and give feedback on your submissions!
