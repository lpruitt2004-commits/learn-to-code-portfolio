# FAQ & Troubleshooting

## Common Questions

**Q: My JavaScript code isn’t running in the browser. What should I check?**
- Make sure your <script> tag is correctly linked in index.html.
- Check the browser console for errors (F12 or right-click > Inspect > Console).

**Q: Git won’t let me push. What do I do?**
- Did you run git add and git commit first?
- Is your branch tracking the remote? Try git push --set-upstream origin main.

**Q: My CSS changes aren’t showing up.**
- Did you save style.css?
- Is the <link rel="stylesheet"> tag correct in index.html?
- Try hard-refreshing the browser (Ctrl+Shift+R).

## Error Fixes
- Syntax errors: Check for missing brackets, parentheses, or typos.
- Permission errors: Use sudo if needed, or check file permissions.
- Merge conflicts: Use git status and resolve conflicts in your editor.
