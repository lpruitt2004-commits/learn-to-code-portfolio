# Glossary: Backend Terms

- **API:** Application Programming Interface
- **CRUD:** Create, Read, Update, Delete
- **Endpoint:** A URL where an API can be accessed
- **Middleware:** Code that runs between request and response in Express
- **JWT:** JSON Web Token, used for authentication
- **ORM:** Object-Relational Mapping, e.g., Sequelize, Mongoose
- **Schema:** Structure of a database table or collection
- **Route:** URL pattern handled by server code
