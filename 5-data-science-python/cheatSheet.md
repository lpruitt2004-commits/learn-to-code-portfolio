# Cheat Sheet: Data Science & Python

## Pandas
- Import: `import pandas as pd`
- Read CSV: `df = pd.read_csv('file.csv')`
- Filter: `df[df['col'] > 10]`

## Matplotlib
- Import: `import matplotlib.pyplot as plt`
- Plot: `plt.plot(x, y)`
- Show: `plt.show()`

## Scikit-learn
- Import: `from sklearn.linear_model import LinearRegression`
- Fit: `model.fit(X, y)`
