# FAQ & Troubleshooting: Data Science & Python

## Common Questions

**Q: ImportError: No module named pandas**
- Did you install pandas? Run `pip install pandas`.

**Q: Plot not showing up?**
- Did you call `plt.show()` in matplotlib?

**Q: Jupyter notebook issues?**
- Try restarting the kernel or clearing output.
