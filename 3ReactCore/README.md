# React Core Assignments

## Getting Started
- Use Create React App or Vite to scaffold a new React project in this folder.
- Practice with JSX, components, props, state, and hooks.

## Example Assignments
- Build a counter app
- Create a dynamic list with add/remove
- Form with validation

## Instructions
1. Scaffold a React app: `npx create-react-app react-core` or use Vite.
2. Complete assignments in the app's `src` folder.
3. Use AI prompts in your code comments for help.
