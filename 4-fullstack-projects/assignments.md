# Assignments: Fullstack Projects

## 1. MERN Stack App
- Build a CRUD app using MongoDB, Express, React, and Node.js.
- Deploy your app to a cloud platform (Heroku, Vercel, etc.).

## 2. RESTful Blog
- Create a blog with RESTful API endpoints and a React frontend.
- Add user authentication and comments.

## 3. Auth & API Integration
- Build an app that uses OAuth or JWT for authentication.
- Integrate with a third-party API (e.g., weather, news, or maps).

## 4. Capstone/Portfolio Project
- Design and build a project of your choice that showcases your fullstack skills.
- Prepare a README and deployment instructions.

> Place your code and documentation in the relevant project folders. CLxL2 can review and give feedback on your submissions!
