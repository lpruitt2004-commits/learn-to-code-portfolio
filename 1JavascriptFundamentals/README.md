
# JavaScript Fundamentals Assignments

## Step-by-Step Learning Path
1. **variables.js** – Practice variables, data types, and operators.
2. **functions.js** – Learn about functions, scope, and closures.
3. **arrays-objects.js** – Work with arrays, objects, and destructuring.
4. **es6.js** – Explore modern ES6+ JavaScript features.
5. **async.js** – Understand callbacks, promises, and async/await.
6. **dom.js** – Manipulate the DOM and handle events.
7. **main.js** – Complete integrated assignments (Calculator, To-Do List, DOM Manipulation).

## Assignments (in main.js)
- Calculator
- To-Do List
- DOM Manipulation

## Instructions
1. Follow the order above for each file.
2. Open `index.html` in your browser to test your code.
3. Style your page using `style.css`.
4. Use AI prompts in your code comments for help.
