# Cheat Sheet: Frontend Quick Reference

## HTML
- Boilerplate: `<!DOCTYPE html> ... <html> ... </html>`
- Link CSS: `<link rel="stylesheet" href="style.css">`
- Script tag: `<script src="main.js"></script>`

## CSS
- Select by class: `.myClass {}`
- Select by id: `#myId {}`
- Media query: `@media (max-width: 600px) { ... }`

## JavaScript
- Select element: `document.querySelector('p')`
- Add event: `el.addEventListener('click', fn)`
- Change text: `el.textContent = 'Hi'`

## React
- Component: `function MyComp() { return <div /> }`
- useState: `const [x, setX] = useState(0)`
