# Glossary: Frontend Terms

- **Element:** A single part of an HTML document.
- **Selector:** A pattern used in CSS to select elements.
- **Component:** A reusable piece of UI in React.
- **State:** Data that changes over time in a React component.
- **Event:** An action (like a click) that JS can respond to.
- **Flexbox:** A CSS layout mode for arranging items in a row or column.
- **Grid:** A CSS layout system for 2D layouts.
- **JSX:** A syntax extension for JavaScript used in React.
