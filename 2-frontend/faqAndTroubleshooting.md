# FAQ & Troubleshooting: Frontend

## Common Questions

**Q: My CSS isn’t applying to my HTML. What should I check?**
- Is the <link rel="stylesheet"> tag correct and in the <head>?
- Did you save your CSS file?
- Try a hard refresh (Ctrl+Shift+R).

**Q: My JavaScript isn’t working.**
- Is your <script> tag at the end of the body or using defer?
- Check the browser console for errors.

**Q: React component not rendering?**
- Is React and ReactDOM loaded?
- Are you using correct JSX syntax?

## Error Fixes
- Syntax errors: Check for typos and missing brackets.
- Console errors: Read the error message and line number.
