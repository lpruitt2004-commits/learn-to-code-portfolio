# Portfolio Site

## Project Overview
Build a personal portfolio site using React or vanilla JS. Showcase your projects, skills, and contact info.

## Suggested Files
- index.html
- style.css
- main.js (or React src/)

## Features
- About section
- Projects section
- Contact form

## Instructions
1. Scaffold your project files.
2. Use AI prompts in your code comments for help.
3. Deploy your site (Netlify, Vercel, or GitHub Pages).
