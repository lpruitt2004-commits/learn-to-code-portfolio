# CRUD App

## Project Overview
Build a CRUD (Create, Read, Update, Delete) app such as a task manager using React or vanilla JS.

## Suggested Files
- index.html
- style.css
- main.js (or React src/)

## Features
- Add, edit, delete, and list items
- Persistent storage (localStorage or backend)
